# 可用的脚本

在项目中，你可以运行这些命令：

## `npm start`

运行这个脚本会启动服务，自动打开默认浏览器展示你的页面。当你重新编辑代码后，页面还会自动刷新。

## `npm run build`

运行这个脚本将会编译你的项目，你可以在项目中的 dist 目录中找到编译后的文件用于部署。

## `npm run analyze`

analyze 脚本做的事情与 build 的相同，但是他会打开一个页面来展示你的依赖信息。如果需要优化性能和包大小，你需要它。

## `npm test`

这个脚本会执行一系列测试，包括 e2e 测试。
